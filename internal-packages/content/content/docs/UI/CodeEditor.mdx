title: Code Editor
---

Code editor is the part of both [Squiggle Playground](/docs/UI/Playground) and [Squiggle Editor](/docs/UI/Editor).

It is implemented with [CodeMirror](https://codemirror.net/).

## Features

- Syntax highlighting
- Error highlighting
- Autocompletion
- Tooltips for computed values
- Tooltips for inferred types
